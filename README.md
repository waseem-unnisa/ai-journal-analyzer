# AI Journal Analyzer

An AI-powered journal analyzer that detects your mood, 
identifies key themes, and gives personalized suggestions 
to improve your day.

## Tech Stack
- Python
- Google Gemini API
- google-generativeai library

## How to Run

1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Add your Gemini API key in journal.py
4. Run: python journal.py

## Example Output
- Mood Analysis: Detects emotions like stress, anxiety, happiness
- Key Themes: Identifies what topics are on your mind
- Suggestions: 3 kind, practical tips to improve your day
